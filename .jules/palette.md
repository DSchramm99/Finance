# Palette's Journal - UX & Accessibility Learnings

## 2025-04-11 - Progress Bars and Tooltips for Financial Metrics
**Learning:** Visualizing technical scores (Trend, Risk) using progress bars significantly reduces the cognitive load for users compared to raw numerical data. Adding technical tooltips to column headers provides essential context for technical indicators (like ATR-based stops) without cluttering the UI.
**Action:** Prioritize visual scoring and technical context as high-impact micro-UX improvements in financial dashboards.

## 2025-04-11 - Consistency in UI Language
**Learning:** Even when user-facing data (like index names or certain labels) is in a local language (German), core UI headers should maintain consistency with the established language of the codebase (English) unless a full localization is explicitly requested.
**Action:** Be cautious with unsolicited translations; focus on enhancing existing patterns first.
